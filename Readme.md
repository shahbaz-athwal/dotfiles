# Dotfiles

This repository contains my personal dotfiles for macOS and Windows(WSL).

## For Mac

### Express Install

```bash
xargs brew install --cask < casks.txt
xargs brew install < formulae.txt
```

### Stow

```bash
stow git zsh karabiner ghostty
```
